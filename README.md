# WXCL

WXCL 是一种中间代码，通过专门的翻译程序将其转换为微信小程序代码。  
这种设计允许开发者用统一的语法编写代码，专注于业务逻辑，而不必直接编写平台相关的代码，从而提高开发效率，降低维护成本。

---

## 特性

- **中间代码设计**  
  使用统一、简洁的语法，减少多平台开发的重复劳动。

- **自动转换**  
  提供高效的翻译工具，将中间代码转换为标准的 wx 小程序代码。

- **易于调试和维护**  
  中间代码层使代码逻辑更加清晰，方便调试和后续扩展。

- **扩展性**  
  除了 wx 小程序，未来还可以支持更多平台的代码转换。
